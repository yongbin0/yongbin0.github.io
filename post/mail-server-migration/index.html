邮件服务器迁移godaddy

@yb · Jun 23, 2009 · 1 min read

CR要把邮箱从目前的香港 Pachosting迁移到Godaddy，又要晚上做。好不容易MX记录生效了，Godaddy的exchange plan创建邮箱一直在pending，其客服解释来解释去去还是其FAQ里的内容一样，也没说生效需要24小时，浪费我的越洋电话，Pachosting的客服也没见好到哪去。一夜未果，只能恢复原状，一晚上也就改了个DNS server和MX记录，还好修改了MX优先级，下次直接配邮箱好了再怎么说Pachosting在邮件服务器的hosting上也相当专业，为哪门子气非要迁到US，价格也没什么优势，性能也不见得好，难道这是他们的爱国主义
