Smplaer播放mp4格式文件的问题

@yb · Feb 6, 2010 · 1 min read

电脑里有几个mp4格式和3gp格式的文件，用smplayer播放的时候很不流畅，用mplayer来播放却正常，这就排除了视频驱动和vpdau的问题看到网上说可以尝试一下三种办法：1. 在smplayer的属性 - 高级设置里把 correct pts 改成 no2. 把 demuxer 改成mov3. 在属性 - 性能里把 cache for local files设置为 0 最终生效的是第三种办法，再把cache改成0后，Smplayer也可以流畅的播放mp4文件了
